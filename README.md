## Installation 🔧

To run this project locally, follow these steps:

1. Clone the repository :
   ```bash
   git clone https://github.com/DeepLeau/portfolio.git
   ```
2. Access the project directory :
   ```bash
   cd portfolio
   ```
3. Install dependencies :
   ```bash
   npm install
   ```
4. Start the development server :
   ```bash
   npm start
   ```
